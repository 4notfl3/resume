03项目作品

AI-Chatroom 实时聊天系统

2026.3 - 至今

全栈实时通信落地：基于 FastAPI WebSocket + Nuxt 3 实现多人聊天室，设计 8 类 JSON 消息协议，支持广播/私聊/系统通知/在线用户实时同步。MCP Agent 聊天集成：基于 pydantic-ai + MCP StreamableHTTP 实现 LLM Agent 编排层，动态注入远端 MCP Server 工具（天气查询），跑通「用户输入 → LLM 推理 → MCP 工具调用 → 流式回显」完整 Agent 闭环，支持 Qwen/DeepSeek/GLM 多模型热切换及流式增量打字机效果。前端工程化落地：基于 Nuxt 3 Composables 模式实现单一 useChat 状态管理中心（WebSocket 生命周期 + 消息分桶 + 频道切换），落地 MongoDB 消息持久化、自动重连（指数退避 1-10s）、30s 心跳保活、私聊 Toast 通知等生产级特性。

本地私有知识多模型 AI Agent 对话系统

2026.5 - 至今

全栈 Agent 推理闭环：基于 FastAPI SSE 流式 + Nuxt 3 实现多轮 AI 对话，跑通「用户输入 → LLM 推理 → MCP StreamableHTTP 工具调用（TTL 缓存 + 中文工具名安全映射）→ 流式回显」完整 Agent 闭环，支持 Qwen/DeepSeek/GLM/Kimi 多模型热切换及打字机效果。本地 RAG 知识库引擎：基于 ChromaDB + all-MiniLM-L6-v2 搭建文档向量检索引擎，落地上传自动分块、余弦相似度检索、检索结果自动注入对话上下文增强回答，异常静默降级不阻塞主链路。前端工程化落地：基于 Nuxt 3 实现单一状态管理中心（认证/聊天/会话/API 四大模块）、SQLite WAL 持久化、附件分类上传、MCP 广场三级降级策略、Nginx 反向代理部署等生产级方案。
